关于开展2017年实验室开放项目申报工作的通知
来源：   更新日期：2017-7-11 14:54:07  浏览次数：2371
各本科教学单位：
    为了充分发挥实验室的资源优势，进一步提高实验室利用率，为更多学生提供自主发展和实践锻炼的空间，学校继续组织进行实验室开放工作。现将2017年实验室开放项目申报工作通知如下：
    1.实验室开放范围为全校不同层次、不同功能的所有本科教学实验室。全校所有本科教学实验室在完成正常教学的同时，要充分发掘潜力、优化资源配置，创造条件实施开放。
    2.开放项目应有明确的实验内容和可考核的项目成果，项目内容有价值、有创意，项目应有利于实验教学水平的提高，应有利于学生动手能力和创新思维的培养。
    3.项目指导教师必须在校在岗，指导教师在项目期间出外进修的须终止项目。每个开放项目可参与学生人数5-8人，项目周期为一年。 
    4.开放项目主要资助范围  
    （1）虚拟仿真类实验系统开发；实验教学设备自制；实验室安全教育及考试系统开发等（可根据不同学院、专业等信息进行分类考试）；  
    （2）切合学科新发展的综合性、设计性、创新性实验项目设计；
    （3）学生自主设计的开放项目，项目内容需具有一定的综合性、设计性、创新性；
    （4）学生基础技能拓展类项目。 
    5.本学年实验室开放项目拟立约60项。 
    6.开放项目申报经费不超过2000元，经费使用需按照申报书预算严格执行。获资助的项目须依托实验室现有设备，报销经费只限于开放项目进行过中产生的图书资料费、耗材费等，不列支其他费用。 
    7.实验室开放项目申报流程
    各有关单位布置申报工作，组织实验教师及实验技术人员填写《太原理工大学实验室开放项目申请表》→实验中心组织对所报开 放项目申报内容的真实性、开放项目可行性等情况进行审核→经学院审批同意后将申报的开放项目报教务处→学校组织相关人员对开放项目进行审定并公示→正式立项的开放项目由所在单位组织实施。。
    8.项目验收
    各单位在开放项目完成后组织结题答辩，需对开放项目的软、硬件成果依据成果的创新性、实用性、实际价值等方面进行考核。 
    结题后学院需将开放项目的相关材料（学生实验报告或研究论文、实验室开放项目记录表、实验室开放项目成绩表、实验耗材登记表等相关资料）汇总存档备查，并按照学校要求提交相关资料。
    请各单位积极组织教师进行开放项目申报，于2017年9月5日前将填写好的《太原理工大学实验室开放项目立项申请表》（附件1）、《太原理工大学实验室开放项目申报情况汇总表》（附件2）及电子稿报送至教务处实践教学管理科（迎西校区办公主楼235室）。
